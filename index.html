Fonctionnalités

Tout ce dont vous avez besoin

Conçu pour les professionnels de l'impression grand format et de la découpe vinyle.

Placement automatique

L'algorithme optimise la disposition de vos objets sur le rouleau pour minimiser les espaces vides et réduire la chute matière.

Catalogue de matières

Base de données intégrée des largeurs standards des principales marques de vinyle (Oracal, Avery, 3M…). Sélection en un clic.

Rapport de perte

Visualisez la surface utilisée, la chute réelle et le pourcentage de perte par groupe d'objets dans un rapport détaillé.

Rotation intelligente

Autorisez la rotation à 90° des objets pour trouver la disposition la plus compacte. Option indépendante pour les textes.

Groupement par calque

Organisez vos objets en plans de travail distincts par calque, couleur ou en masse. Idéal pour séparer les références clients.

Mode plaque

En plus du mode rouleau, optimisez le placement sur des plaques de dimensions fixes pour la découpe CNC ou laser.

Installation

Prêt en 2 minutes

Aucune compétence technique requise. Un double-clic suffit.

1

Téléchargez OPTI

Cliquez sur le bouton de téléchargement ci-dessous pour obtenir le fichier .zip. Décompressez-le où vous le souhaitez.

2

Lancez l'installeur

Double-cliquez sur INSTALLER OPTI.command. Si macOS bloque le fichier : clic droit → Ouvrir → Ouvrir quand même. Choisissez "1) Installer OPTI".

3

Ouvrez Illustrator

Ouvrez (ou relancez) Adobe Illustrator. Dans le menu : Fenêtre → Extensions → OPTI. Le panneau s'affiche dans votre espace de travail.

4

Optimisez !

Sélectionnez vos objets dans Illustrator, choisissez votre matière dans le catalogue, et cliquez sur ▶ Optimiser.

FAQ

Questions fréquentes

Le panneau n'apparaît pas dans le menu Extensions

Vérifiez que le mode debug CEP est bien activé — le script d'installation le fait automatiquement. Relancez complètement Adobe Illustrator (pas juste fermer le panneau). Si le problème persiste, relancez le script d'installation et sélectionnez "1) Installer OPTI" à nouveau.

macOS affiche "extension non signée" ou bloque le script

C'est normal — OPTI n'est pas distribué via l'Adobe Exchange, ce qui déclenche un avertissement de sécurité standard. Pour le script d'installation : clic droit → Ouvrir → Ouvrir quand même. Pour l'extension dans Illustrator : allez dans Réglages Système → Confidentialité et sécurité et cliquez "Ouvrir quand même".

OPTI fonctionne-t-il sur Windows ?

La version actuelle est macOS uniquement. Une version Windows est en cours de développement. Revenez sur cette page ou suivez le projet pour être notifié à sa sortie.

Ma marque de vinyle n'est pas dans le catalogue

Utilisez l'option Largeur personnalisée dans l'onglet Catalogue pour saisir n'importe quelle largeur en millimètres. Vous pouvez aussi contacter pour suggérer l'ajout d'une marque dans une prochaine mise à jour.

Comment désinstaller OPTI ?

Relancez le fichier INSTALLER OPTI.command et choisissez "2) Désinstaller OPTI". L'extension est supprimée proprement sans laisser de traces.

Comment savoir si une mise à jour est disponible ?

Le panneau OPTI vérifie automatiquement les mises à jour à chaque ouverture d'Illustrator. Si une nouvelle version est disponible, une bannière apparaît en haut du panneau avec un lien pour télécharger la mise à jour.
